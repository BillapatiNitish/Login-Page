# Login Page

A simple and responsive Login Page built using **HTML** and **CSS**. This project was created to practice front-end development and improve my skills in designing clean and user-friendly web pages.

## Features

* Clean and modern UI
* Responsive design
* Email and password input fields
* Login button
* Easy to customize

## Technologies Used

* HTML5
* CSS3

## Project Purpose

The main purpose of this project is to practice creating a basic login page using HTML and CSS. It helped me understand page layout, form design, styling, and responsive design techniques.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your browser.

## Future Improvements

* Add JavaScript form validation
* Show/Hide password feature
* "Forgot Password" option
* Connect the login page to a backend database
